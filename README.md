Welcome to my project!

This is a chatbot built using Python, leveraging the power of AI to engage in natural-sounding conversations.
The bot is designed to be a fun and informative companion, and I'm excited to share it with you.
Key Features:

• **Login System**: Remember your chat history and pick up where you left off
• **New Chat Option**: Start a new conversation any time
• **Logout Button**: Securely log out of your account
• **Grok's API Integration**: Engage in chat with the bot, using Twitter's Grok's API

How to Use:
Simply download or clone this repository from GitHub
Run the script to start the chatbot
Use the login system to create an account and get started
